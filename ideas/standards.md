# Standards

## Concept

Idea files should contain concepts for future features

## Files

Filename should be saved as \[commit author\]-\[month_day_year\]-\[commit number that day\]

## File Structure

File should start with the header "\[commit author\] \[month/day/year\] \[commit number that day\]"

Each subheader should be named the concept's name, and below should be a list of the contents of the concept

Example:

```
# TEST 1/1/1970 1

## A cool concept

A concise description of the concept
```
